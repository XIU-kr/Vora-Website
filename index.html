특징

프로 도구를, 설정 없이.

무거운 런타임도, 클라우드 로그인도 없습니다. 다운로드해서 바로 작업을 시작하세요.

AI 복원

Big-LaMa 인페인팅으로 지우고 싶은 대상을 자연스럽게 제거합니다. 배경과 질감을 인식해 한 번의 클릭으로 깔끔하게.

프로급 선택 도구

매직 완드·라소·마퀴와 Invert / Expand / Contract / Feather. 선택 영역에 지우기·채우기·배경 교체·복원까지 바로 적용합니다.

논파괴 레이어

12가지 블렌드 모드, 5가지 레이어 이펙트, 곡선·레벨·색상 균형 등 조정 레이어를 원본 훼손 없이 적용합니다.

기능

필요한 도구는 전부 들어 있습니다.

드로잉 & 페인팅

  • 펜 · 도형(사각/원/선/화살/베지어)
  • 클론 스탬프 · 힐링 브러시 · 히스토리 브러시
  • 그라디언트 레이어 생성
  • [ / ] 브러시 크기 · D/X 색상 제어

이미지

  • 이미지 크기 · 캔버스 크기(9-앵커) · 자동 트림
  • 90°/180°/270° 회전, 좌우·상하 반전
  • 자동 향상 · 자동 레벨 · 샤픈
  • 노이즈 추가 · 색상 교체

편집

  • Fill (Shift+F5) · Stroke (inside / center / outside)
  • Copy Merged (Ctrl+Shift+C)
  • 이름 있는 실행 취소/다시 실행
  • 선택 영역에 붙여넣기 (Ctrl+Shift+V)

레이어 시스템

  • 정렬 & 분포 (가로·세로 · 좌/중/우 · 상/중/하)
  • 정돈 — 맨 앞/뒤로, 앞/뒤로 보내기 (Ctrl+Shift+])
  • 레이어 마스크 · 12 블렌드 모드
  • 5 레이어 효과 — 그림자 · 스트로크 · 내부 글로우 · 컬러/그라디언트 오버레이

조정 레이어

  • 밝기/대비 · 레벨 (Ctrl+L) · 곡선 (Ctrl+M)
  • 색조/채도 (Ctrl+U) · 색상 균형
  • 모든 조정은 논파괴 레이어로 적용

프로젝트 & 내보내기

  • .vora 프로젝트 파일 — 전체 페이지·레이어를 하나의 JSON으로
  • PNG · JPG · WebP · 멀티페이지 PDF · 편집 가능한 PPTX
  • 배치 내보내기 · 1×–8× 스케일
  • .vora 파일 확장자 연결 — 더블클릭으로 바로 열기

작업 흐름

  • 멀티페이지 프로젝트 (필름스트립)
  • 커맨드 팔레트 (Ctrl+K) · 퀵마스크 (Q)
  • 눈금자·그리드·가이드 · 100% 줌 (Ctrl+1)
  • 자동 저장 · 다음 실행 시 복원 프롬프트
  • OS 언어 자동 감지 (한국어 / 영어)
